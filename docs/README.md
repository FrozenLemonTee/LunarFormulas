# Documentation

LunarFormulas follows the same documentation shape as LunarUnits: the root
README introduces the public surface, `examples/` contains runnable cookbook
tests, and this directory records design notes that should stay stable as the
library grows.

Current documents:

- [`architecture.md`](architecture.md): package structure, dependency boundary
  and formula evaluation model.

Planned documents:

- `roadmap.md`: future domains such as electromagnetism, thermal, signal and
  economics formulas.
- `cookbook.md`: longer examples once more domain packages exist.
